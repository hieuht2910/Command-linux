# Config multi user remote control desktop #
--------------------------------------------------

## Cài đặt Nhiều user có thể kết nối đến win10 ##

Giải nén file .zip trong thư mục window
chạy file install
Chạy file RDPConf
Tích vào ô Single session per user trong khung General Settings
Tại mục "Authentication Mode" chọn Network Level Authentication
Tại mục "Session Shadowing Mode" chọn Full access with user's permission
